Explore o Universo da Programação

Descubra o propósito e o poder de cada linguagem de código.

Aprenda de forma simples e visual: veja exemplos de códigos “Olá Mundo” e entenda a função de cada linguagem.

JavaScript

Usada para tornar páginas web interativas e dinâmicas, controlando eventos e elementos do navegador.

console.log("Olá Mundo!");

Python

Popular por sua simplicidade. Muito usado em inteligência artificial, automação, ciência de dados e web.

print("Olá Mundo!")

Java

Usada em sistemas grandes, aplicativos Android e servidores. É estável e multiplataforma.

public class Main {
  public static void main(String[] args) {
    System.out.println("Olá Mundo!");
  }
}

C#

Desenvolvida pela Microsoft, é usada em programas Windows e jogos feitos na Unity.

using System;
class P { static void Main(){ Console.WriteLine("Olá Mundo!"); } }

C++

Usada para criar jogos e sistemas de alto desempenho, como sistemas operacionais.

#include <iostream>
int main(){ std::cout << "Olá Mundo!"; return 0; }

Assembly

Pode ser usada para tarefas de baixo nível que exigem controle total do hardware.

nasm -f elf32 ola.asm -o ola.o
